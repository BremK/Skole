MathMl

MathML er en markup language som representere matematiske formler på en strukturert måte. Det finnes to typer av MathML; C-MathML og P-MathML. For mer informasjon om MathML klikk Les mer.

SVG

SVG er en 2D vektorbasert bildeformat. Vi kan bruke SVG til å lage logoer, ikoner, animasjoner eller skjemaer. Man kan manipulere SVG manuelt, med CSS eller Javascript. For mer informasjon om SVG klikk Les mer.
