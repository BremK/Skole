Git og GitHub

Git/github er en distribuert versjonskontrollsystem. Det er en måte å dele opp koder, holde oversikt over endringer, dokumenter og liknende.
Noen fordeler ved å bruke git/github

  1. Godt egna for samarbeid. Si hvilke endringer man har gjortm når filen ble oppretta
  2. Strukturert lagra versjoner
  3. Gjenopprette tidligere versjoner
  4. Oversikt over historien
  5. Backup
